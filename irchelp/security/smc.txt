                                <p>Configuring SMC Router for IRC Use (identd and DCC)</p>

<p>This is part of IRChelp.org&#8217;s Firewall FAQ:
    http://www.irchelp.org/irchelp/security/fwfaq.html</p>

<p>Based upon http://duxcw.com/dcforum/dcboard.cgi?az=printer_format&amp;forum=DCForumID2&amp;om=1852&amp;omm=4</p>

<p>Slight edits by Jolo 6/6/02</p>

<hr />

<h1>4, RE: Irc dcc send and SMC7004ABR</h1>

<p>Posted by ChAoTiC on May-29-02 at 10:48 AM
In response to message #0</p>

<p>1) First, configure the following mIRC options:</p>

<p>file | options | connect | options
   Default server port : 6667
      (if necessary, change 6667 to whatever the network requires like 7000)
   click on advanced button
   check the 2nd checkbox &#8220;Binds socket to ip&#8221;, in the text box
      write your external ip adress
      (Don&#8217;t know it? Try //USERHOST $me or //DNS $me, then check status window.)</p>

<p>file | options | connect | localinfo
   delete all text in Local Host and IP Address
   check &#8220;always get local host&#8221;
   select the radio button for lookup method &#8220;server&#8221;</p>

<p>file | options | dcc | options
   DCC port: first 1024, last 1028
      (that opens 5 ports which is enough for most people)</p>

<p>That&#8217;s all for the mIRC configuration.</p>

<p>2) In the router:</p>

<p>Now you actually need to open and forward ports in your router&#8217;s firewall.
This lets the router correctly send incoming traffic from the outside to your
PC.</p>

<p>In the Virtual Server page (setup | advanced settings | virtual server),
open the following ports, note that the private and public ports are the same.</p>

<p>Port 59 is the DCC server, 113 is for identd, 6667 (or whatever you set the
default server port to in mIRC earlier) is for IRC itself. And finally the
range of ports 1024 and up are for DCC sending, you can add more if you like,
just make sure you match it with what you put in mIRC&#8217;s DCC options.</p>

<p>private port   type   public port
    59          tcp      59
   113          tcp     113
  6667          tcp    6667
  1024          tcp    1024
  1025          tcp    1025
  1026          tcp    1026
  1027          tcp    1027
  1028          tcp    1028</p>

<p>For each line, put in the private IP of your PC where you want to use IRC.
If you don&#8217;t know your private IP, /run winipcfg. You may need to select
your network adapter from the popup text box on top.</p>

<p>3) You&#8217;re done. Disconnect from the server and reconnect. If you did it right
you can check to see if identd is working by doing &#8220;/whois nickname&#8221; on
yourself, see if the ~ (tilde) before your username is gone. Then try to 
DCC send some small text file to a buddy.</p>
