                                <h2>MSchv32.exe trojan horse</h2>

<p>by Allaik 22-06-98</p>

<h2>Description</h2>

<p>This trojan is just like the router32b. It loads itself to memory on
startup.  When you run this trojan, it will copy itself to
c:\windows\system\MSchv32.exe.  Then it will modify your windows&#8217;s
registry. This makes the program run each time you run windows. </p>

<h2>Check</h2>

<p>To see if you are infected with MSchv32.exe trojan, press ctrl-alt-del
to open up the &#8220;Close Program&#8221; window. It will show a list of programs
that you are running. Search for a program called MSchv32.exe. If it
exists, you are infected. </p>

<h2>Instruction on removal</h2>

<ol>
<li><p>Press ctrl-alt-del. Open up the &#8220;Close Program&#8221; window. Select the
program called MSchv32.exe and click &#8220;End Task&#8221;. Wait about 10
seconds, and windows will ask you that u confirm with closing 
program. Press &#8220;End Task&#8221;. </p></li>
<li><p>Delete the file C:\windows\system\MSchv32.exe. (Using your Explorer
or DOS) </p></li>
<li><p>Go to Start Menu -> Run. Type &#8220;regedit.exe&#8221;
First, it would be wise to backup the registry file
Click Registry Menu -> Export Registry File
FileName: C:\windows\BackupRegistry.reg
Export Range: All
Then press Ok to save.</p></li>
<li><p>Follow the path: HKEY_USERS -> .DEFAULT -> Software -> Microsoft 
-> Windows -> CurrentVersion -> Run
You will see a key that contain the Data &#8220;C:\windows\system\MSchv32.exe&#8221;
Select it, and press delete -> click &#8220;Yes&#8221; to confirm</p></li>
<li><p>Run your Mirc Client, press Alt-R. 
Go to View Menu, see if you can find a file called script.ini. 
If you have it, select it.   Press Find Text and enter &#8220;opshit&#8221;</p>

<p>If you find the word, &#8220;opshit&#8221;, look at the filename under the text box. 
Remember the path and filename. For example: C:\script.ini or C:\mirc\script.ini
Then go to  File Menu  and press Unload.</p></li>
<li><p>Press ok to exit to Mirc Client again. 
In any window, press /remove c:\script.ini or the path you found on 
the filename in previous step.</p></li>
</ol>

<hr />

<p>You have now uninstalled that trojan. Good luck!</p>

<p>For more information on other trojan:
http://www.irchelp.org/irchelp/security/</p>
