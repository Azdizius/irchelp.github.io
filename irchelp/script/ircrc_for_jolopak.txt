                                <h1><em>#</em>#<em>##</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em></h1>

<h1>*                                          *</h1>

<h1>*   sample .ircrc setup file for JoloPak   *</h1>

<h1>*                                          *</h1>

<h1><em>#</em>#<em>##</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em>#<em>#</em></h1>

<h2>last updated: 10/22/00 with JoloPak 1.4</h2>

<h2>This file doesn&#8217;t change with each JoloPak because it is just an example</h2>

<h2>of an .ircrc setup file which lets you install or upgrade to the latest</h2>

<h2>jolopak.scr with no changes, instead putting all custom changes here.</h2>

<h2>Many lines are commented out, delete the leading #&#8217;s to activate a line.</h2>

<h2>INSTALLATION (3 easy steps):</h2>

<h2>1. Save the file as .ircrc in your shell account home directory.</h2>

<h2>Also put the main JoloPak file jolopak.scr in the same place.</h2>

<h2>(If you have older versions of either file, back them up first!)</h2>

<h2>2. Personalize &#8220;myversion&#8221;, &#8220;myhome&#8221; and &#8220;myuserinfo&#8221; below.</h2>

<h2>3. If you installed ircII yourself, verify the LOAD<em>PATH and HELP</em>PATH.</h2>

<h2>Now Launch ircII as usual (usually type &#8220;irc&#8221;) &#8212; that&#8217;s it!</h2>

<h2>Put your home directory or where you saved jolopak.scr if different</h2>

<h2>after &#8216;myhome&#8217; below. To find your home directory, at the UNIX prompt</h2>

<h2>type &#8220;echo $HOME&#8221;, or &#8220;cd&#8221; followed by &#8220;pwd&#8221;.</h2>

<p>assign myhome /home/jyl</p>

<h2>Change &#8220;ircII&#8230;&#8221; below to whatever version you are running. To find</h2>

<h2>your version, launch IRC by &#8220;irc -q&#8221;, then /CTCP your_nick VERSION, copy</h2>

<h2>everything after &#8220;CTCP VERSION reply from your_nick:&#8221;. When in doubt,</h2>

<h2>don&#8217;t tamper with this, and don&#8217;t get cute or you risk getting /kill&#8217;d.</h2>

<p>assign myversion ircII 2.8.2 SunOS 5.8</p>

<h2>The next long line is returned to anybody doing /ctcp your_nick userinfo</h2>

<h2>This is your chance to be creative - don&#8217;t mess with other CTCP responses!</h2>

<h2>This is my actual userinfo, just to show you an example. :)</h2>

<p>assign myuserinfo Jack Smith, I enjoy Broadway musicals and violent movies, please see my web site at http://blah</p>

<h2>Now that you&#8217;ve customized your ircII properly, load the JoloPak script</h2>

<h2>Don&#8217;t change this line or move it elsewhere!!!</h2>

<p>eval LOAD $myhome/jolopak.scr</p>

<h2>That&#8217;s pretty much it, anything after this is purely optional!</h2>

<h2>From here on you can add any further customizations. All of these</h2>

<h2>examples are commented out. To activate them, delete the leading &#8220;#&#8221;</h2>

<h2>symbol from each line beginning with just 1 &#8220;#&#8221;.</h2>

<h2>If you have problems with /load, uncomment and change the following</h2>

<h2>to suit your needs.  Here&#8217;s what to do if the default ircII directories</h2>

<h2>change from /usr/local/ to the &#8220;myhome&#8221; directory that you set earlier.</h2>

<h1>eval SET LOAD_PATH .:$myhome:$myhome/script:$myhome/lib/irc/script</h1>

<h1>eval SET HELP_PATH $myhome/lib/irc/help</h1>

<h2>If you do change LOAD_PATH above, uncomment the following 2 lines</h2>

<h2>so that you reload these standard scripts using your correct LOAD_PATH</h2>

<h1>LOAD tabkey</h1>

<h1>LOAD killpath</h1>

<h2>We&#8217;re all done now. You can add customizations at the end of this file,</h2>

<h2>or make a new file like &#8220;Private&#8221; and use it by uncommentng the next line:</h2>

<h1>eval LOAD $myhome/Private</h1>

<h2>All the following lines are examples of how to customize JoloPak, you</h2>

<h2>may toggle their action by commenting or uncommenting the lines.</h2>

<h2>if uncommented, turn off auto recording of /msg&#8217;s when you are away</h2>

<h1>^assign record<em>when</em>away NO</h1>

<h2>if uncommented, turn off the auto /away if you idle for 15 min</h2>

<h1>ON ^IDLE -15</h1>

<h2>if uncommented, enable delayed auto_rejoin after you get kicked. Note</h2>

<h2>some channels consider auto rejoin to be annoying and you may get banned!</h2>

<p>autorejoin on</p>
